Usage guidelines

  • Use --accent-500 for primary CTAs and interactive links.
  • Use --surface for cards and list surfaces; reserve --bg for page backgrounds.
  • Use chips with --chip-bg for suggested labels to provide subtle visual grouping.
  • Maintain contrast ratio >= 4.5:1 for body text over background.
